📋 Step 1 — Upload Manifest

Required columns: Order Number, Name, Phone, Pax, Pickup Time, Pickup Location, Agent, Driver, Bus#

📖 How to use — Morning Pickup Reminder
  1. Export the manifest from Rezdy for today's tours and save as .xlsx.
  2. Do not remove or rename any header row. The following columns must be present and spelled exactly:
    Order Number  ·  Name  ·  Phone  ·  Pax  ·  Bus#  ·  Driver  ·  Pickup Time  ·  Pickup Location  ·  Agent
    Missing or renamed columns will cause the upload to fail.
  3. Click Upload & Preview — guests are grouped by pickup location for easy review.
  4. Use the location toggle buttons to select or deselect an entire pickup group at once.
  5. Use the Select all / Deselect all buttons to manage all rows, or check individual rows to include or exclude specific guests.
  6. Guests highlighted in blue are duplicates — already sent today. They are unchecked by default but can be re-selected if needed.
  7. Default send mode is SMS Only. Switch to SMS + Email or Email Only if needed.
  8. Click Send to Selected to send reminders only to checked guests.
⚠️ Never delete or overwrite the header row in the Excel file — this will break the upload and no reminders will be sent. ⚠️Error Handling Network Error: Usually caused by temporary network congestion. Wait 1–2 minutes, refresh the page, then upload and send again. Internal Server Error: Stop sending immediately. Take a screenshot and notify Annie. Use alternative channels for urgent communications. Non-urgent orders should wait until the issue is resolved. Most system issues are fixed within 2 hours.
