Tour Confirmation — Send

General Order Confirmation

Required columns: Order Number, First Name, Last Name, Email, Customer Phone, Pick-up Time, Pick-up Location

📖 How to use — Tour Confirmation
  1. Download the manifest from Rezdy for the tour date.
  2. Select the matching Tour Type and Tour Date above.
  3. Upload the .xlsx file — the filename should match the tour and date (e.g. bryce-zion-bus-2026-06-01.xlsx).
  4. Click Upload & Preview to review the guest list before sending.
  5. Any orders already sent will be flagged as Already Sent — skip or override as needed.
  6. Choose SMS + Email, SMS Only, or Email Only, then confirm send.
⚠️ Never delete or overwrite the header row in the Excel file — this will break the upload and no reminders will be sent.
⚠️ Error Handling
  • Network Error: Usually caused by temporary network congestion. Wait 1–2 minutes, refresh the page, then upload and send again.
  • Internal Server Error: Stop sending immediately. Take a screenshot and notify Annie. Use alternative channels for urgent communications. Non-urgent orders should wait until the issue is resolved. Most system issues are fixed within 2 hours.

⚡ Last Minute Order

For orders placed within the cancellation window — skip reconfirmation, send directly to lunch selection & pickup details.

Required columns: Order Number, First Name, Last Name, Email, Customer Phone, Pick-up Time, Pick-up Location

📖 How to use — Last Minute Order
  1. Use this section only for orders placed within the cancellation window (typically same-day or next-day bookings).
  2. These guests skip the reconfirmation step and go directly to lunch selection & pickup details.
  3. Select the Tour Type and Tour Date, then upload the manifest.
  4. Review the preview and confirm send.
⚠️ Do not use this for regular bookings — guests will not receive the standard confirmation & reconfirmation flow.
⚠️ Error Handling
  • Network Error: Usually caused by temporary network congestion. Wait 1–2 minutes, refresh the page, then upload and send again.
  • Internal Server Error: Stop sending immediately. Take a screenshot and notify Annie. Use alternative channels for urgent communications. Non-urgent orders should wait until the issue is resolved. Most system issues are fixed within 2 hours.
